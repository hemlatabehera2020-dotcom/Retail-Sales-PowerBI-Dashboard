# Retail Sales Performance Dashboard

## Project Overview
This project analyzes retail sales data using Power BI.  
The dashboard provides insights into sales performance, profit trends, regional analysis, and future sales forecasting.

## Tools Used
- Power BI
- DAX
- Power Query
- Data Visualization

## Dashboard Features
- KPI Metrics (Sales, Profit, Quantity)
- Monthly Sales Trend Analysis
- Category & Subcategory Performance
- Regional Sales Map
- Customer Segment Analysis
- Forecasting (Next 6 Months)

## Key Business Insights
- South region generates the highest sales
- Furniture category contributes the highest revenue
- Technology products have strong profit margins
- Discounts impact profitability
